# Project: vagrant-boxes
---

Several vagrant boxes I use for development

# Installation

```
$ git clone https://github.com/qnimbus/vagrant-boxes
$ cd vagrant-boxes/base
$ vagrant up
```

## Usage

```
$ vagrant ssh
```

and when done with the VM, either

```
$ vagrant suspend
```

or

```
$ vagrant halt
```

or

```
$ vagrant destroy
```
